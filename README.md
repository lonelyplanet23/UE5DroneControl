# DroneControl

Developed with Unreal Engine 5

## 方案：
两台飞机：
+ 真实飞机：接受真实的飞机信号: 位置信息，按照真实飞机的飞行轨迹飞行
+ 虚拟飞机：接受用户的控制信号：键盘，手柄点击移动，并发送UDP信号给真实飞机
+ 虚拟飞机和真实飞机的坐标系一致：单位 厘米


## 目前功能
1. 绑定无人机模型，更换移动动画/待机动画。
2. 虚拟飞机：
   1. 在鼠标点击逻辑处添加发送UDP信号（时间戳：xyz坐标）
   2. W/S控制上下移动，（z轴）注意高度限制，并发送UDP信号
   3. 配置 drone network: 需要发送飞控主机端的ip地址，端口
4. 真实飞机：（有一定延迟）
    1. 不断接收无人机控制主机端传来的UDP信号，解析时间戳，xyz坐标
    2. 当接受的信号值发生变化时，飞向xyz坐标点
    3. 配置Real Time Config： 端口
5. 地图白模，贴图，接入Cesium Google API接口
