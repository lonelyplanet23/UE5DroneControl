# DroneControl

Developed with Unreal Engine 5

## 方案：
两台飞机：
+ 真实飞机：接受真实的飞机信号: 位置信息，按照真实飞机的飞行轨迹飞行
+ 虚拟飞机：接受用户的控制信号：键盘，手柄点击移动，并发送UDP信号给真实飞机
+ 虚拟飞机和真实飞机的坐标系一致：单位 米
**因此，白模上需要有长度的参考，知道鼠标的位置**
