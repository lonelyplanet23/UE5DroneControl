> 版本：v0.2
> 日期：2026-04-22（依据接口与通讯数据规范 v1.0 更新）
> 说明：本文档结合当前代码实现状态编写。已实现且符合架构的功能已在需求表中标注，存在偏差的功能同样列出，由执行人员检查修改。

## 一、前端架构概览

```
UE5 前端
│
├── 通讯模块（待实现）
│   ├── HTTP Client（注册/查询/删除，轮询后端）
│   └── WebSocket Client
│       ├── 发送：控制指令（目标点、暂停、阵列任务）
│       └── 接收：遥测推送（位置/姿态/电量/状态/告警）
│
├── 地图交互模块（部分实现）
│   ├── 射线检测选中无人机（已实现：DroneOpsPlayerController）
│   ├── 左键点击地图生成目标点（已实现）
│   └── Cesium GPS → UE 世界坐标转换（待实现）
│
├── 无人机状态模块（待实现）
│   ├── 注册表单 UI（填写后 POST 到后端）
│   ├── 无人机列表渲染（轮询后端 GET）
│   └── 删除/编辑操作（PUT/DELETE 到后端）
│
├── 实时部署模块（部分实现）
│   ├── 影子机（Shadow Drone）
│   │   ├── 按指令轨迹插值移动（已实现：RealTimeDroneReceiver VInterpTo）
│   │   └── 延迟计算（待实现）
│   ├── 镜像机（Mirror Drone）
│   │   ├── 接收后端推送遥测（当前：UDP直连；目标：WebSocket）
│   │   ├── Cesium 坐标系轴向对齐（待验证/实现）
│   │   └── 驱动 Mesh 位置/姿态（已实现，需切换数据源）
│   ├── 实时控制模式（部分实现）
│   │   ├── 射线检测选中（已实现）
│   │   ├── 点击地图生成目标点（已实现）
│   │   └── 发送控制指令（当前：UDP；目标：WebSocket）
│   ├── 阵列执行模式（待实现）
│   │   ├── 无人机槽位映射 UI
│   │   ├── 下发阵列任务到后端（HTTP POST）
│   │   ├── 集结阶段可视化
│   │   │   ├── 镜像机跟随遥测实时渲染（正常运行，无需特殊处理）
│   │   │   ├── 影子机隐藏或跟随镜像机同步移动（集结期间不独立运动）
│   │   │   └── 弹窗提示集结进度（"集结中 N/M 架已就位"）
│   │   └── 接收执行状态反馈（WebSocket：assembling / assembly_complete / assembly_timeout）
│   └── 弹窗通知系统（待实现，统一调用入口）
│       ├── 告警弹窗（低电量 / 断联）
│       ├── 集结状态弹窗（集结中进度 / 集结完成 / 集结超时）
│       └── 通用提示弹窗（操作反馈 / 错误提示）
│
├── 镜头控制系统（已实现）
│   ├── 跟随镜头（跟随选中无人机）
│   ├── 自由镜头（WASD/QE + 鼠标旋转）
│   └── F 键切换 + 平滑过渡
│
├── 阵列编辑模块（待实现）
│   ├── 航点编辑（俯视点击 + 高度调节）
│   ├── 执行模式配置（侦察/巡逻/攻击）
│   ├── 阵列 JSON 序列化/持久化
│   ├── 阵列预演动画
│   └── 碰撞检测（UE 物理/几何检测）
│
└── 初始化位置同步（锚定流程，待实现）
    ├── 接收后端"上电"事件（WebSocket）
    ├── 请求 GPS 锚点（HTTP GET /anchor）
    ├── Cesium API 转 UE 世界坐标
    └── 设置镜像机/影子机锚点
```

## 二、关键类说明

| 类名 | 文件 | 职责 |
|------|------|------|
| `ADroneOpsPlayerController` | DroneOps/Control/ | 主控制器：输入处理、镜头切换、选中/点击派发 |
| `ARealTimeDroneReceiver` | 根目录 | 镜像机：UDP接收、YAML解析、插值移动、遥测推送 |
| `AMultiDroneCharacter` | 根目录 | 影子机/发送机：UDP发送、选中状态、命令发送 |
| `UDroneRegistrySubsystem` | DroneOps/Core/ | 全局注册表：无人机描述符、遥测缓存、选中状态 |
| `UDroneTelemetryComponent` | DroneOps/Drone/ | 遥测组件：缓存快照、广播更新、推送到注册表 |
| `UDroneSelectionComponent` | DroneOps/Drone/ | 选中组件：视觉高亮（点光源/文字标签/材质叠加） |
| `UDroneCommandSenderComponent` | DroneOps/Drone/ | 命令发送：构建32字节UDP包，发送到后端控制器 |
| `ICoordinateService` | DroneOps/Core/ | 坐标服务接口：WorldToNed / NedToWorld / GPS转换 |
| `USimpleCoordinateService` | DroneOps/Core/ | 简单坐标服务（非Cesium，线性近似） |
| `ADroneOpsGameMode` | DroneOps/Control/ | 游戏模式：初始化坐标服务、注册表、Pawn接管 |
