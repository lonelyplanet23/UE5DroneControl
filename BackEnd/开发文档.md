## 一、技术选型

| 项目 | 选型 | 说明 |
|------|------|------|
| 语言 | Python 3.13 | 环境已就绪，无需额外安装 |
| Web 框架 | FastAPI + uvicorn | 原生支持 HTTP REST 和 WebSocket，异步高性能 |
| 数据校验 | Pydantic v2 | 与 FastAPI 深度集成 |
| 持久化 | JSON 文件 | 轻量，路径由 config.yaml 配置 |
| UDP 通信 | Python socket（标准库） | 无额外依赖 |

> 注：架构设计文档建议 Go/Rust/C++，但当时环境中 Python 方案最先落地，后续已切换为当前的 C++ 后端。

## 三、启动方式

```bash
pip install -r requirements.txt
cd BackEnd
python -m uvicorn app.main:app --host 0.0.0.0 --port 8080 --reload
```

服务启动后：

- HTTP REST：`http://localhost:8080`
- WebSocket：`ws://localhost:8080/ws`
- Swagger：`http://localhost:8080/docs`
