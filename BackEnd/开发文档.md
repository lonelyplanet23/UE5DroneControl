# 后端开发文档

> 版本：v0.1
> 日期：2026-04-22
> 面向：后端开发工程师、测试人员

---

## 一、技术选型

| 项目 | 选型 | 说明 |
|------|------|------|
| 语言 | Python 3.13 | 环境已就绪，无需额外安装 |
| Web 框架 | FastAPI + uvicorn | 原生支持 HTTP REST 和 WebSocket，异步高性能 |
| 数据校验 | Pydantic v2 | 与 FastAPI 深度集成 |
| 持久化 | JSON 文件 | 轻量，路径由 config.yaml 配置 |
| UDP 通信 | Python socket（标准库） | 无额外依赖 |

> 注：架构设计文档建议 Go/Rust/C++，但当前环境未安装 Go，Python 已就绪且 FastAPI 满足高并发 WebSocket 推送需求，后续如需切换语言，接口协议不变。

## 二、目录结构

```
BackEnd/
├── app/
│   ├── main.py          # FastAPI 应用入口，WebSocket /ws 端点
│   ├── config.py        # config.yaml 加载器
│   ├── models.py        # Pydantic 数据模型
│   ├── registry.py      # 无人机注册表（内存 + JSON 持久化）
│   ├── ws_manager.py    # WebSocket 连接管理器（广播/单发）
│   ├── udp_stub.py      # UDP 控制包发送存根（24字节小端格式）
│   └── routers/
│       ├── drones.py    # /api/drones 路由
│       ├── arrays.py    # /api/arrays 路由
│       └── debug.py     # /api/debug 路由（debug=true 时启用）
├── config.yaml          # 运行参数配置
├── requirements.txt     # Python 依赖
├── run.bat              # Windows 启动脚本
└── run.sh               # Linux/Mac 启动脚本
```

---

## 三、启动方式

```bash
# 安装依赖（首次）
pip install -r requirements.txt

# 启动服务（Windows）
run.bat

# 或手动启动
cd BackEnd
python -m uvicorn app.main:app --host 0.0.0.0 --port 8080 --reload
```

服务启动后：
- HTTP REST：`http://localhost:8080`
- WebSocket：`ws://localhost:8080/ws`
- 接口文档（Swagger）：`http://localhost:8080/docs`

## 四、已实现内容（第一周：基础架构搭建）

### 4.1 HTTP REST 接口（全部实现，功能为存根）

| 接口 | 方法 | 状态 | 说明 |
|------|------|------|------|
| `/api/drones` | GET | ✅ | 返回注册无人机列表 |
| `/api/drones` | POST | ✅ | 注册新无人机，含重名/槽位冲突/上限校验 |
| `/api/drones/{id}` | PUT | ✅ | 更新无人机信息 |
| `/api/drones/{id}` | DELETE | ✅ | 删除无人机，释放槽位 |
| `/api/drones/{id}/anchor` | GET | ✅ | 返回 GPS 锚点（由遥测注入填充） |
| `/api/arrays` | POST | ✅ stub | 接收阵列任务，返回 array_id，执行引擎待实现 |
| `/api/arrays/{id}/stop` | POST | ✅ stub | 停止阵列任务，执行引擎待实现 |

### 4.3 WebSocket（/ws）

| 方向 | 消息类型 | 状态 |
|------|----------|------|
| UE → 后端 | `move` | ✅ 解析 drone_id + UE 偏移，转换后入队 |
| UE → 后端 | `pause` | ✅ 支持 drone_ids 数组 |
| UE → 后端 | `resume` | ✅ 支持 drone_ids 数组 |
| 后端 → UE | `telemetry` | ✅ 由 inject 接口触发广播 |
| 后端 → UE | `event` (power_on) | ✅ 由 inject 接口触发 |
| 后端 → UE | `alert` (low_battery) | ✅ 含去重逻辑 |
| 后端 → UE | `assembling` | stub（执行引擎待实现） |
| 后端 → UE | `assembly_complete` | stub |
| 后端 → UE | `assembly_timeout` | stub |

### 4.4 坐标转换（已实现）

- NED(m) → UE 偏移(cm)：`x×100, y×100, z×(-100)`
- UE 偏移(cm) → NED(m)：`x×0.01, y×0.01, z×(-0.01)`

### 4.5 持久化

- 注册数据写入 `./data/drones.json`，服务重启后自动恢复

### 4.6 UDP 存根

- `app/udp_stub.py`：实现 24 字节小端控制包格式（`<dfffi`），`send_control()` 和 `send_heartbeat()` 接口已定义，可直接对接真实无人机

## 六、测试方法

### 快速验证（Apifox / curl）

```bash
# 注册无人机
curl -X POST http://localhost:8080/api/drones \
  -H "Content-Type: application/json" \
  -d '{"name":"drone1","model":"DJI","ip":"192.168.1.10","port":14550,"slot_number":1}'

# 注入遥测（触发 WebSocket 推送）
curl -X POST http://localhost:8080/api/debug/drone/d1/inject \
  -H "Content-Type: application/json" \
  -d '{"position":[1.0,2.0,-10.0],"q":[1,0,0,0],"velocity":[0,0,0],"battery":85,"gps_lat":39.9,"gps_lon":116.3,"gps_alt":50.0}'

# 查看接口文档
open http://localhost:8080/docs
```

### WebSocket 测试（wscat）

```bash
npm install -g wscat
wscat -c ws://localhost:8080/ws

# 发送 move 指令
{"type":"move","drone_id":"d1","x":1000,"y":2000,"z":500}

# 发送 pause
{"type":"pause","drone_ids":["d1"]}
```
